# CSS Learning Assignments

This repository contains my **CSS practice assignments** completed during my learning journey with the **Elzero Web School CSS course**.

The purpose of these assignments is to improve my skills in styling web pages and understanding how CSS works in real-world layouts.

---

## Topics Covered

During these assignments I practiced many important CSS concepts including:

- CSS Selectors
- Colors and Backgrounds
- Borders and Shadows
- Box Model
- Positioning
- Pseudo Classes & Pseudo Elements
- CSS Variables
- CSS Counters
- Transitions and Hover Effects
- Flexbox Layout
- Responsive calculations using `calc()`
- Form styling

---

## Assignments Included

The repository includes multiple CSS practice assignments such as:

- Styled elements with shadows and borders
- Hover animations and transitions
- CSS counters for numbering elements
- Custom UI components
- Flexbox layout systems
- Navigation and page layout design
- Styled forms and inputs
- Gradient decorations and pseudo elements

Each assignment focuses on applying a specific CSS concept.
